A simple open-source Python tool that organizes messy folders automatically.

## Features

* Organizes files by type (Images, Videos, Documents, etc.)
* Dry-run preview mode
* Beginner-friendly and open for contributions

## Installation

```bash
git clone https://github.com/YOUR_USERNAME/file-organizer-cli.git
cd file-organizer-cli
python organizer.py --help
```

## Usage

```bash
python organizer.py ~/Downloads
```

Preview changes without moving files:

```bash
python organizer.py ~/Downloads --dry-run
```
